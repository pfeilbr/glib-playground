# glib-playground

learn [GNOME | GLib](https://wiki.gnome.org/Projects/GLib)

see [`main.c`](main.c)

## Prerequisites

```sh
brew install cmake
brew install glib
brew install pkg-config
```

## Build and Run
```sh
mkdir -p build && cd build
cmake .. && make

# run
./glib-playground
```
